Changelog

2.1

  • Added Second Order Mutation
  • Added 2 new Standard Operators : Deletion Operators
  • Added Code Coverage Detection
  • Added a possibility to modify the time after which mutant is marked as killed due to a timed-out
  • Fixed several small bugs
  • All new functionalities can be turned on or off in Option View

2.0

  • Limited scope mutation testing - an easy way to run the process in a context of one method.
  • Major performance improvements - extensive caching.
  • Parallel mutant processing.
  • Fixes to stability and correctness.

1.5

  • Changed the extension (additional selected structural and object-oriented mutation operators) developmment framework to faciliate CCI
  • Performance improvements - mutants are created on demand.
  • Various fixes.

1.0

  • Creation and testing of mutants.
  • Ability to choose mutation operators to include in current session.
  • Original and mutated code comparison (C#, IL).
  • Ability to save created mutants to the hard drive.
  • Included operators for ASP.NET MVC framework.
